# 支援的使用案例

Adobe Experience Platform Web SDK可搭配其他Adobe解決方案(例如Adobe Analytics、Audience Manager、Target和Experience Platform)，提供多種個人化使用案例。

如需實作支援使用案例的詳細資訊，請參閱 [Web SDK使用案例專案頁面](https://github.com/orgs/adobe/projects/18/views/1).
