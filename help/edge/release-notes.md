# 發行說明

## 2.1.0 版

* 支援IAB 2.0同意標準。
* 支援在命令中傳遞其他 `setConsent` ID。
* 支援覆蓋 `datasetId` 命令中 `sendEvent` 的。
* 移除命 `syncIdentity` 令並支援在命令中傳遞這些 `sendEvent` ID。
* 支援合金顯示器(詳[細資訊)](https://github.com/adobe/alloy/wiki/Monitoring-Hooks)
* 傳遞 `environment: browser` 實作詳細內容資料。
