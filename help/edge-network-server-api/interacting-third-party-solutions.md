# 與第三方解決方案交互

## 總覽 {#overview}

使用邊緣網路伺服器API的事件轉發功能將收集到的資料發送到非Adobe解決方案。

## 配置資料流事件轉發 {#event-forwarding}

要使第三方解決方案能夠從伺服器API接收資料，您需要 [配置資料流](../edge/fundamentals/datastreams.md#event-forwarding-settings) 用於事件轉發。
