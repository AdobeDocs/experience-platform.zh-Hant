# 使用API建立串流連線

需要註冊串流連接，才能開始將資料串流至Adobe Experience Platform。 在註冊串流連線時，您需要提供一些關鍵詳細資訊，例如串流資料來源。

在註冊串流連線後，身為資料產生者的您將擁有可用來將資料串流至平台的唯一URL。

若要瞭解如何建立已驗證的串流連線，請閱讀[HTTP API來源API指南](../../sources/tutorials/api/create/streaming/http.md)。
