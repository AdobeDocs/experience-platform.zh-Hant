# 使用API建立流連接

需要流連接註冊才能啟動流資料到Adobe Experience Platform。 註冊流連接時，需要提供一些關鍵詳細資訊，如流資料源。

註冊流連接後，作為資料生成者，您將擁有一個唯一的URL，該URL可用於將資料流傳輸到平台。

要瞭解如何建立經過身份驗證的流連接，請閱讀 [HTTP API源API指南](../../sources/tutorials/api/create/streaming/http.md)。
