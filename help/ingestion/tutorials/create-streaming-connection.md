# 使用API建立串流連線

若要開始將資料串流至Adobe Experience Platform，需要串流連線註冊。 註冊串流連線時，您需要提供一些重要詳細資訊，例如串流資料的來源。

註冊串流連線後，您（身為資料製作者）會有一個唯一URL，可用來將資料串流至Platform。

若要了解如何建立已驗證的串流連線，請參閱 [HTTP API來源API指南](../../sources/tutorials/api/create/streaming/http.md).
