# 使用UI建立流連接

為了啟動流資料到 [!DNL Experience Platform]，必須首先建立流HTTP連接。 在建立流連接時，您需要提供關鍵詳細資訊，如流資料源，以及您是否打算從受信任的（已驗證）源或不受信任的（未驗證）源發送資料。

註冊流連接後，您將擁有一個唯一的URL，該URL可用於將資料流傳輸到 [!DNL Platform]。

要瞭解如何建立流連接，請閱讀 [HTTP API源UI指南](../../sources/tutorials/ui/create/streaming/http.md)。
