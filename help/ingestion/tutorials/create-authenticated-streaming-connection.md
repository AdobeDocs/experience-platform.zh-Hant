# 建立驗證的串流連線

「已驗證的資料收集」可讓[!DNL Real-time Customer Profile]和[!DNL Identity]等Adobe Experience Platform服務區隔來自受信任來源和不受信任來源的記錄。 想要傳送個人識別資訊(PII)的客戶可以透過傳送存取Token作為POST要求的一部分來傳送。

若要瞭解如何建立已驗證的串流連線，請閱讀[HTTP API來源API指南](../../sources/tutorials/api/create/streaming/http.md)。
