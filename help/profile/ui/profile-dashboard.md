# [!UICONTROL 設定檔] 儀表板 {#profile-dashboard}

Adobe Experience Platform使用者介面(UI)提供了一個儀表板，您可以通過該儀表板檢視有關您的解決方案的重要資訊。 [!DNL Real-Time Customer Profile] 資料，在每日快照期間擷取。

如需如何存取及與互動的詳細說明， [!UICONTROL 設定檔] UI中的儀表板，以及若要深入瞭解儀表板中顯示的可用量度，請造訪 [[!UICONTROL 設定檔] 儀表板指南](../../dashboards/guides/profiles.md).

如需Experience Platform使用者介面中所有儀表板功能的概觀，請先閱讀 [儀表板概觀](../../dashboards/home.md).
