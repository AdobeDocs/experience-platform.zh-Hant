# [!UICONTROL 設定檔]儀表板 {#profile-dashboard}

Adobe Experience Platform使用者介面(UI)提供儀表板，您可以透過儀表板檢視有關您的[!DNL Real-Time Customer Profile]資料的重要資訊，如每日快照期間所擷取。

如需有關如何在UI中存取及與[!UICONTROL 設定檔]儀表板互動的詳細指示，以及深入瞭解儀表板中顯示的可用量度，請造訪[[!UICONTROL 設定檔]儀表板指南](../../dashboards/guides/profiles.md)。

如需Experience Platform使用者介面中所有儀表板功能的概觀，請先閱讀[儀表板概觀](../../dashboards/home.md)。
