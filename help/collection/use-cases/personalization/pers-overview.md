# Personalization使用案例概觀

Adobe Experience Platform Web SDK可讓企業為其Web屬性提供各種個人化使用案例。 您可以在各種Adobe解決方案(例如Target、Journey Optimizer和Audience Manager)間，順暢地連線資料收集、決策和啟動程式。

SDK支援靈活的架構，包括使用者端、伺服器端和混合模型，因此能夠擷取和呈現符合大多陣列織使用案例的個人化選件。

## 網頁SDK範例

除了此資料夾中的檔案頁面外，Adobe還會維護您可參考的範例應用程式存放庫。 如需更多個人化使用案例，請參閱GitHub上的[網頁SDK範例](https://github.com/adobe/alloy-samples/)，包括：

* 使用者端個人化
* 伺服器端個人化
* 混合個人化
* 使用單頁應用程式的Personalization
