#  Currencydata類型

 Currency是描述貨幣量的標準XDM資料類型，包括貨幣類型和轉換日期。

![](../images/data-types/currency.png)

| 屬性 | 資料類型 | 說明 |
| --- | --- | --- |
| `amount` | 雙倍 | 顯示可表示的顏色數。 |
| `conversionDate` | DateTime | 顯示可表示的顏色數。 |
| `currencyCode` | 字串 | 顯示可表示的顏色數。 |
