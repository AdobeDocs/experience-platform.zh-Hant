# Mixin名稱更新

數個「體驗資料模型」(XDM)混合名稱已變更。 由於這些混合的欄位結構保持不變，因此僅更改名稱不應影響使用這些混合的任何現有模式。 但是，為了在Adobe Experience Platform使用者介面中查看和使用這些混音，必須使用其新名稱。
