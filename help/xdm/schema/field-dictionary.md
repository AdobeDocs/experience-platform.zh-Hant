# XDM欄位字典


[!DNL Experience Data Model] (XDM)定義了一組標準欄位，可在合成Adobe Experience Platform結構時使用。 使用這些欄位可讓 [!DNL Platform] 服務，因為使用XDM的所有服務都會明確了解這些欄位提供的特定定義。

請參閱 [綱要構成基本知識](composition.md) 如需結構構建區塊的詳細資訊，包括定義結構的原則和最佳實務 [!DNL Experience Platform].

下表包含欄位名稱（並非所有XDM欄位都是以「xdm」進行命名），以及其標題和說明、類型和格式（視需要而定），以及其對應的XDM類型和可找到欄位的資源（結構、結構欄位群組和資料類型）。

您可以在 [!DNL Schema Editor] 若要檢視這些資源的詳細資訊，請參閱 [結構編輯器教學課程](../tutorials/create-schema-ui.md#browse).

>[!NOTE]
>
>如果欄位被標籤為「**已棄用**「，這表示它不應再用於新實作。 欄位仍可供回溯相容性使用，但未來版本將不再支援。
