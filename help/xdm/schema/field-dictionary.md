# XDM欄位字典


[!DNL Experience Data Model] (XDM)定義了一組標準欄位，這些欄位在合成Adobe Experience Platform架構時可以使用。 使用這些欄位會導致跨 [!DNL Platform] 服務，因為使用XDM的所有服務都將明確瞭解這些欄位提供的特定定義。

查看 [架構組合基礎](composition.md) 有關架構構建塊的詳細資訊，包括定義架構的原則和最佳做法 [!DNL Experience Platform]。

下表包括欄位名稱（並非所有XDM欄位都使用「xdm」命名），以及其標題和說明、類型和格式（如果需要），以及相應的XDM類型和可在其中找到該欄位的資源（方案、架構欄位組和資料類型）。

可以在 [!DNL Schema Editor] 查看這些資源的詳細資訊，如 [架構編輯器教程](../tutorials/create-schema-ui.md#browse)。

>[!NOTE]
>
>如果欄位已標籤為「」**已棄用**&#x200B;即不再用於新實施。 該欄位仍可用於向後相容，但將來的版本中將不再支援該欄位。
