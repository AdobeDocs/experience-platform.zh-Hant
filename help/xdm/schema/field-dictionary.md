# XDM欄位字典


[!DNL Experience Data Model] (XDM)定義了一組標準欄位，可在構成Adobe Experience Platform結構描述時使用。 使用這些欄位會導致[!DNL Platform]服務間的一致行為，因為所有使用XDM的服務都會明確瞭解這些欄位提供的特定定義。

請參閱結構描述組合的[基本概念](composition.md)，以取得有關結構描述建置區塊的詳細資訊，包括定義[!DNL Experience Platform]的結構描述的原則和最佳實務。

下表包括欄位名稱（並非所有XDM欄位都使用「xdm」進行名稱空間），以及欄位的標題和說明、型別和格式（如果需要），以及其對應的XDM型別和可找到欄位的資源（結構描述、結構描述欄位群組和資料型別）。

您可以使用[!DNL Schema Editor]中的搜尋功能來檢視這些資源的詳細資訊，如[結構描述編輯器教學課程](../tutorials/create-schema-ui.md#browse)中所示。

>[!NOTE]
>
>如果欄位標示為&quot;**DEPRECATED**&quot;，表示它不應該再用於新的實作。 此欄位會繼續提供回溯相容性，但未來版本將不再支援此欄位。
