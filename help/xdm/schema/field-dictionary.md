# XDM欄位字典


[!DNL Experience Data Model] (XDM)會定義在構成Adobe Experience Platform結構描述時可使用的一組標準欄位。 使用這些欄位會導致以下專案中的行為一致： [!DNL Platform] 服務，因為所有使用XDM的服務都會明確瞭解這些欄位提供的特定定義。

請參閱 [結構描述組合的基本面](composition.md) 如需結構描述建置區塊的詳細資訊，包括定義結構描述的原則和最佳實務 [!DNL Experience Platform].

下表包括欄位名稱（並非所有XDM欄位都使用「xdm」進行名稱空間），以及欄位的標題和說明、型別和格式（如果需要），以及其對應的XDM型別和可找到欄位的資源（結構描述、結構描述欄位群組和資料型別）。

您可以使用中的搜尋功能 [!DNL Schema Editor] 以檢視這些資源的詳細資訊，如 [結構描述編輯器教學課程](../tutorials/create-schema-ui.md#browse).

>[!NOTE]
>
