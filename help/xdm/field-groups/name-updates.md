# 架構欄位組名稱更新

數個Experience Data Model(XDM)架構欄位群組的名稱已變更。 由於這些欄位組的欄位結構保持不變，僅更改名稱不應影響採用這些欄位組的任何現有架構。 但是，為了在Adobe Experience Platform使用者介面中查詢及使用這些欄位群組，必須使用其新名稱。
