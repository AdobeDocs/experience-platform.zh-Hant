# 架構欄位組名稱更新

幾個「體驗資料模型」(XDM)架構欄位組的名稱已更改。 由於這些欄位組的欄位結構保持不變，因此僅更改名稱不應影響使用這些欄位組的任何現有方案。 但是，為了在Adobe Experience Platform用戶介面中查找和使用這些欄位組，必須使用它們的新名稱。
