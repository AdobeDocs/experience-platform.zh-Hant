# 臨機結構

在特定情況下，可能需要建立Experience Data Model(XDM)架構，其中欄位的名稱僅限單一資料集使用。 這稱為「臨機」架構。 臨機結構描述用於數種資料擷取工作流程，包括擷取CSV檔案和建立特定類型的來源連接器。 如需臨機結構描述的詳細資訊，包括如何建立結構描述的步驟，請參閱臨 [機結構描述教學課程](../tutorials/ad-hoc.md)。
