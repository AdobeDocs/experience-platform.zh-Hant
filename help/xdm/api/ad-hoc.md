# 臨機結構

在特定情況下，可能需要建立(XDM)架構，其中欄位的名稱僅限單一資料集使用。 [!DNL Experience Data Model] 這稱為臨機結構。 臨機結構描述用於數種資料擷取工作流程，包括擷取CSV檔案和建立特定類型的來源連接器。 如需臨機結構描述的詳細資訊，包括如何在 [!DNL Schema Registry] API中建立其中的步驟，請參閱 [臨機結構教學課程](../tutorials/ad-hoc.md)。
