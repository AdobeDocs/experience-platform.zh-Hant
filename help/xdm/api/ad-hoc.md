# 臨時結構描述

在特定情況下，可能需要建立[!DNL Experience Data Model] (XDM)結構描述，其欄位必須由單一資料集命名以供使用。 這稱為臨時結構描述。 特定結構用於數個資料擷取工作流程，包括擷取CSV檔案和建立特定型別的來源聯結器。 如需臨機架構的詳細資訊，包括如何在[!DNL Schema Registry] API中建立架構的步驟，請參閱[臨機架構教學課程](../tutorials/ad-hoc.md)。
