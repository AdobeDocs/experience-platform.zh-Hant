# 匯入端點

此 `/rpc/import` 端點 [!DNL Schema Registry] API可讓您從產生的匯出負載建立Experience Data Model(XDM)資源。 可從兩個來源建立匯出裝載：

* 此 [`/rpc/export` 端點](./export.md) 會從現有XDM資源建立匯出裝載，以便您在沙箱之間共用資源。
* 此 [`/rpc/csv2schema` 端點](./csv-to-schema.md) 從CSV範本建立匯出裝載。

建立匯出裝載後，您可以使用 `/rpc/import` 端點，在您選取的沙箱中產生資源（和所有相依資源）。

## 快速入門

此 `/rpc/import` 端點是 [[!DNL Schema Registry] API](https://www.adobe.io/experience-platform-apis/references/schema-registry/). 繼續之前，請檢閱 [快速入門手冊](./getting-started.md) 如需相關檔案的連結，請參閱本檔案中讀取範例API呼叫的指南，以及成功呼叫任何Experience PlatformAPI所需的必要標頭重要資訊。

此 `/rpc/import` 端點是遠端程式呼叫(RPC)的一部分，這些呼叫由 [!DNL Schema Registry]. 不同於 [!DNL Schema Registry] API、RPC端點不需要其他標題，例如 `Accept` 或 `Content-Type`，且不使用 `CONTAINER_ID`. 而是必須使用 `/rpc` 命名空間，如下方API呼叫所示。

## 匯入資源 {#import}

在您為XDM資源產生匯出裝載後，就可以在向 `/import` 端點將該資源匯入目標組織和沙箱。

**API格式**

```http
POST /rpc/import
```
