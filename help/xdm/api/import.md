# 匯入端點

[!DNL Schema Registry] API中的`/rpc/import`端點可讓您從產生的匯出裝載中建立體驗資料模型(XDM)資源。 匯出裝載可從兩個來源建立：

* [`/rpc/export`端點](./export.md)會從現有的XDM資源建立匯出承載，讓您在沙箱之間共用資源。
* [`/rpc/csv2schema`端點](./csv-to-schema.md)會從CSV範本建立匯出承載。

建立匯出裝載後，您就可以使用`/rpc/import`端點在您選擇的沙箱中產生資源（以及所有相依資源）。

## 快速入門

`/rpc/import`端點是[[!DNL Schema Registry] API](https://www.adobe.io/experience-platform-apis/references/schema-registry/)的一部分。 繼續之前，請先檢閱[快速入門手冊](./getting-started.md)，以取得相關檔案的連結、閱讀本檔案中範例API呼叫的手冊，以及有關成功呼叫任何Experience PlatformAPI所需必要標題的重要資訊。

`/rpc/import`端點是[!DNL Schema Registry]支援的遠端程式呼叫(RPC)的一部分。 與[!DNL Schema Registry] API中的其他端點不同，RPC端點不需要`Accept`或`Content-Type`等其他標頭，也不使用`CONTAINER_ID`。 相反地，他們必須使用`/rpc`名稱空間，如下面的API呼叫所示。

## 匯入資源 {#import}

產生XDM資源的匯出裝載後，您可以在對`/import`端點的POST請求中使用該裝載，將該資源匯入目標組織和沙箱。

**API格式**

```http
POST /rpc/import
```
