# Privacy Service錯誤代碼 {#privacy-service-error-codes}

使用此參考資料來識別Privacy Service工作結果、診斷失敗，以及決定在&#x200B;**Adobe Experience Platform**&#x200B;中送出或監視隱私權工作時的適當後續步驟。 若要瞭解如何建立、提交及監視隱私權工作，請參閱[隱私權工作端點指南](./privacy-jobs.md)或[Privacy Service UI使用手冊](../ui/user-guide.md)。

Privacy Service錯誤碼是穩定的公開合約。 每個錯誤碼會唯一識別故障或完成狀態，您可依賴這些狀態來進行程式化處理和作業工作流程。

建立自動化或監控工作流程時，需遵守下列保證：

* 發佈後錯誤代碼是穩定的。
* 錯誤訊息可能會改變以提高清晰度，但代碼值不會。
* 隨時間推移可能會新增錯誤碼；不會重新利用現有代碼。

使用錯誤碼（而非訊息文字）來實作自動化或決定邏輯。 如需在不依賴輪詢或訊息字串的情況下有效處理隱私權工作、監視工作狀態及處理錯誤的指引，請參閱[Privacy Service最佳實務](../best-practices.md)。

## 錯誤回應格式 {#error-response-format}

Privacy Service會在工作和請求回應中傳回錯誤資訊。 回應包含數值錯誤代碼，且回應裝載中包含可讀取的訊息。

錯誤碼會傳達授權結果。 此訊息提供疑難排解的其他內容。

本檔案說明每個錯誤碼的含意與目的。 如需欄位層級的回應結構描述和要求詳細資料，請參閱[Privacy Service API檔案](https://developer.adobe.com/experience-platform-apis/references/privacy-service/)。

## 錯誤網域 {#error-domains}

錯誤代碼會依功能網域分組，以協助您更快速地診斷問題。

本檔案使用的網域包括：

* **要求驗證**：要求格式錯誤或包含無效值。 如需要求結構和驗證需求，請參閱[隱私權工作端點指南](./privacy-jobs.md)。
* **授權和布建**：您的組織或使用者缺少必要的存取權。 請參閱[管理許可權](../permissions.md)以檢閱角色型許可權需求。
* **識別與適用性**：識別碼或名稱空間不適用於要求。 請參閱隱私權要求的[身分資料](../identity-data.md)，以瞭解支援的身分型別和名稱空間需求。
* **速率限制**：提交磁碟區超過平台限制。 發生此錯誤時，請降低提交率並重試。
* **資料存取與處理**：系統無法存取或處理要求的資料。 請參閱[疑難排解指南](../troubleshooting-guide.md)以瞭解常見原因和修正步驟。
* **加密和金鑰管理**：無法使用必要的加密金鑰。 請參閱[客戶管理的金鑰](../../landing/governance-privacy-security/customer-managed-keys/overview.md)，以取得金鑰存取、設定和復原指南。
* **工作執行狀態**：工作已完全完成、部分完成或失敗。 請參閱[隱私權工作端點指南](./privacy-jobs.md#status-categories)，以取得工作狀態類別及其含義的說明。

>[!NOTE]
>
>網域指派反映的是錯誤碼的意圖，而非內部服務界限。

## 錯誤代碼參考 {#error-code-reference}

下表列出所有公開Privacy Service的錯誤代碼。

| 錯誤代碼 | HTTP狀態 | 標題 | 說明 |
| ---------- | ----------- | ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------| 
| 1000-400 | 400 | 格式設定錯誤 | 指定應用程式的一或多個資料值有格式問題。 檢查工作詳細資訊以取得其他資訊。 |
| 1001-400 | 400 | 未授權 | 您的組織未布建。 請聯絡您的管理員以取得其他資訊。 |
| 1010-400 | 400 | 缺少權限 | 您沒有執行此動作所需的許可權。 請聯絡您的管理員以要求存取權。 |
| 1020-400 | 400 | 上傳和封存失敗 | 上傳和封存存取資料時發生問題。 請上傳存取資料，然後再試一次。 |
| 1021-400 | 400 | 工作失敗 | 根據請求建立的一或多個隱私工作失敗。 如需其他資訊，請檢閱失敗工作的詳細資訊。 |
| 1022-400 | 400 | 資料存取失敗 | 存取指定的資料時發生問題。 如需其他資訊，請檢閱工作詳細資訊。 |
| 1023-400 | 400 | 資料存取失敗 | 存取或定位指定的資料集ID時發生問題。 請確認提供的ID有效，然後再試一次。 |
| 1024-400 | 400 | 意外錯誤 | 發生未預期的錯誤。 如需其他資訊，請檢閱工作詳細資訊。 |
| 1030-400 | 400 | 超出檔案速率限制 | 工作負載超過檔案速率限制。 請降低您的提交率，然後再試一次。 |
| 1040-400 | 400 | 金鑰加密存取失敗 | 無法處理資料，因為資料存放區已加密，且金鑰存取已撤銷。 請連絡您的金鑰儲存庫管理員，以還原對客戶金鑰的存取權。 |
| 6000-200 | 200 | 成功 | 工作已成功完成。 複查工作詳細資訊，以確認已處理的記錄和結果。 |
| 6051-200 | 200 | 未布建 | 您的組織未針對請求的應用程式進行布建。 該請求不適用。 |
| 6052-200 | 200 | 找不到使用者ID | 找不到部分使用者ID，未知的使用者ID不適用於此產品。 請確認使用者ID有效，然後再試一次。 |
| 6053-200 | 200 | 無效的命名空間 | 提供的身分名稱空間對此應用程式無效。 使用系統可識別的名稱空間。 |
| 6054-200 | 200 | 已部分完成 | 已完成適用資料的工作，但部分資料不適用於該請求。 如需其他資訊，請檢閱工作詳細資訊。 |
