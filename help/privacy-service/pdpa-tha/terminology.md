# PDPA（泰國）術語

以下是與泰國《個人資料保護法》(PDPA)相關的關鍵術語定義清單。

| 詞語 | 定義 |
| --- | --- |
| **個人資料** | 直接或間接識別和個人的資訊，不包括與已故人有關的資訊。 |
| **資料控制器** | 有權就收集、使用或披露個人資料作出決定的個人或實體。 |
| **資料處理器** | 根據資料掌控者的命令收集、使用或公開個人資料的個人或實體。 |
| **敏感資料** | 與種族或族裔出身、政治觀點、宗教或哲學信仰、性行為、犯罪記錄、健康資料、殘疾、工會資訊、遺傳資料、生物特徵資料或可能影響資料主體的任何資料有關的任何個人資料。 |
