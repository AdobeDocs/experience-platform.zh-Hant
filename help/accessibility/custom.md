# Experience Platform的自訂協助工具解決方案

Adobe Experience Platform持續強化，以符合各類使用者的需求，並符合全球標準，涵蓋視覺、聽覺、行動能力或其他殘疾人士。 本檔案概述Experience Platform使用者介面中的自訂協助工具解決方案。

## 首頁和UI總覽

Experience Platform使用者介面符合一般文字、圖形和UI元件所需的對比率。 使用者介面的顏色也經過選擇，可支援所有使用者的無障礙功能，包括視障使用者。

在Platform中，可使用指標點選或操作的UI元素也可以使用鍵盤來參與。 這包括左側導覽、影片播放器、表格等。

Experience Platform致力於符合國際無障礙標準，包括網頁內容無障礙指引2.1 A級和AA級，以及網頁無障礙倡議 — 無障礙多樣化網際網路應用程式(WAI-ARIA)網頁標準。

![Adobe Experience Platform UI的首頁。](images/homepage.png)

## 左側導覽

Experience PlatformUI中的左側導覽可使用鍵盤存取，並可在符合協助工具標準的正常、暫留和選取狀態中提供色彩對比。

從首頁畫面，使用者可以Tab鍵進入左側導覽。 選取 **Shift + Tab** 將使用者返回首頁畫面。

![Experience Platform向左導覽。](images/left-navigation-select.png)

聚焦左側導覽， **標籤** 帶使用者展開和收合互動。 展開或收合左側導覽的功能啟用方式 **輸入（退貨）**.

![Experience Platform左側導覽已摺疊。](images/left-navigation-collapse.png)

當左側導覽成為焦點時，向上鍵和向下鍵會導覽至導覽中的每個專案，並持續循環（換言之，直到使用者用索引標籤離開左側導覽時，焦點才會移開）。 選取時會顯示導覽專案的焦點。 目前選取範圍會以反白和粗體文字顯示。 選取左側導覽專案時， **輸入（退貨）** 會在右側面板中開啟選取的UI專案，但是焦點會維持在左側導覽中，直到使用者離開為止。

![Experience Platform左側導覽並選取來源。](images/left-navigation-sources.png)

Platform中的某些功能並未針對所有使用者啟用。 這些專案會顯示在導覽中，但無法選取。 使用鍵盤導覽時，這些專案會在箭頭導覽期間被跳過，且不能使用選取 **輸入（退貨）**.

![無法選取未對使用者啟用的Experience Platform左側導覽區段。](images/left-navigation-sections-disabled.png)

## 內嵌視訊對話方塊

在Experience Platform中，您可以使用鍵盤導覽反白顯示並選取可用的視訊連結，以檢視視訊。 這會在Platform UI中開啟內嵌視訊對話方塊。

![所選元素周圍出現藍色邊框，表示已套用焦點。](images/profile-overview-tab.png)

## 視訊對話方塊鍵盤協助工具

您也可以使用鍵盤導覽內嵌視訊對話方塊。 下表概述內嵌視訊對話方塊可用的完整鍵盤導覽。

| 對話方塊元素 | 鍵盤協助工具 | 說明 |
|---|---|---|
| 播放和暫停 | 標籤<br/>空白鍵 | 使用 **標籤** 以設定播放按鈕的焦點。 **空白鍵** 開始視訊播放和暫停視訊播放。 |
| 清除器 | 標籤<br/>向左鍵<br/>向右鍵 | 影片播放時，請使用 **標籤** 以集中清理程式。 當清除器成為焦點時， **向左鍵和向右鍵** 分別略過前方和後方5秒的視訊播放。 |
| 靜音 | 標籤<br/>空白鍵 | 使用 **標籤** 將焦點設定在靜音音量元素上。 使用 **空白鍵** 將視訊播放設為靜音或取消靜音。 |
| 卷 | 標籤<br/>向左鍵<br/>向右鍵 | 使用 **標籤** 將焦點放在體積塊元素上。 **向左鍵和向右鍵** 分別向上和向下移動磁碟區。 |
| [!UICONTROL 隱藏式字幕] （「副本」） | 標籤<br/>輸入<br/>向上鍵<br/>向下鍵 | **標籤** 至 [!UICONTROL 隱藏式字幕] (「cc」)元素。 使用 **輸入** 以開啟功能表，並 **向上鍵和向下鍵** 以選取註解的語言。 **輸入** 確認您的選取。 |
| [!UICONTROL 品質] | 標籤<br/>輸入<br/>向上鍵<br/>向下鍵 | 使用 **標籤** 聚焦 [!UICONTROL 品質] 元素。 使用 **輸入** 以開啟功能表和 **向上鍵和向下鍵** 以選取視訊品質。 **輸入** 確認您的選取。 |
| 全熒幕 | 標籤<br/>空格鍵或Enter<br/>逸出 | 使用 **標籤** 將焦點放在全熒幕元素上。 使用 **空格鍵或Enter** 以啟動全熒幕檢視。 **逸出** (「esc」)會退出全熒幕模式。 |
| 關閉 | 標籤<br/>空格鍵或Enter | 使用 **標籤** 以聚焦「關閉」按鈕。 使用 **空格鍵或Enter** 鍵以結束視訊對話方塊。 |

>[!NOTE]
>
>在播放期間，您隨時都可以使用Esc鍵來關閉內嵌視訊對話方塊。

## 檔案拖放

在Experience Platform中，所有檔案選取拖放區域都可使用鍵盤存取。 使用 **標籤** 反白顯示 **[!UICONTROL 選擇檔案]** 和使用 **Enter或空白鍵** 若要選取，會叫用作業系統的檔案選取UI。

上傳檔案後，可使用鍵盤導覽刪除圖示，以移除選取的檔案並上傳新檔案。 使用者可以使用 **標籤** 將焦點放在刪除圖示和 **Enter或空白鍵** 以選取它。 移除檔案後， **[!UICONTROL 選擇檔案]** 會自動成為焦點，且可供選取。

或者，如果上傳的檔案格式不正確，則會顯示錯誤圖示以及錯誤訊息和 **[!UICONTROL 選擇檔案]** 按鈕成為焦點且可供選取。

![檔案拖放區域，其中含有錯誤訊息和「選擇檔案」按鈕。](images/drag-and-drop.png)

使用滑鼠選取拖放區域也會叫用檔案選取UI，或者滑鼠使用者可以選取檔案並拖曳至區域以開始上傳。

![當滑鼠使用者將檔案拖放至區域時，檔案拖放區域成為焦點。](images/drag-and-drop-mouse-over.png)

## 表格瀏覽

Experience Platform使用者介面中的所有表格都可使用鍵盤存取。 您可以透過一系列鍵盤快速鍵，瀏覽表格列和欄並與之互動：

* 在表格標題中，使用 **向下鍵** 以瀏覽表格。 透過導覽時可選取表格標題 **標籤**，您也可以使用變更排序順序 **空白鍵**.
* **向上鍵和向下鍵** 在表格中的列間上下移動。
* 選取列或成為焦點時，使用 **輸入** 列上的右側欄中提供詳細資訊。
* 選取列或成為焦點時，使用 **方向鍵** 以瀏覽列中的每個專案。
* 使用 **輸入** 以選取列中的專案。 如果必須開啟新視窗，則會提醒具有熒幕閱讀器的使用者。
* 當您縮放至200%或更多時，您可以看到 **邊欄檢視窗** 圖示隨著右側邊欄摺疊以提供更多表格檢視空間。

![當使用者縮放至200%時，邊欄檢視窗圖示會聚焦。](images/rail-inspector.png)

### 瀏覽表格鍵盤協助工具

| 鍵盤協助工具 | 說明 |
|---|---|
| HOME （函式+向左鍵） | 聚焦列時，會將使用者導向列中的第一個專案 |
| END （函式+向右鍵） | 聚焦列時，會將使用者導向列中的最後一個專案 |
| 向上翻頁 | 在表格中往上遍歷10列（每頁） |
| 向下翻頁 | 在表格中往下遍歷10列（每頁） |
| 控制+首頁 | 移至表格中的第一列 |
| 控制+結束 | 每頁前往表格中的第一個工作 |

## 結構描述編輯器UI

結構描述編輯器UI可供下列功能存取：

* 結構編輯器支援鍵盤導覽，包括使用 **標籤** 以導覽UI元素。
* **標籤** 輸入搜尋欄位，然後進入結構描述樹狀結構。
* 結構描述樹支援使用方向鍵來瀏覽結構描述樹UI
   * **向上和向下箭頭** 可用來周遊樹狀結構。
   * **左右箭頭** 可用來展開和收合節點，或是在結構描述樹狀目錄上的內嵌動作之間移動。
* **輸入（退貨）** 啟動右側詳細資訊面板中的個別節點詳細資訊。
* 此 **首頁** 索引鍵會傳回樹狀結構的頂端。
* 此 **結束** 鍵會導覽至樹狀結構底部。
* 結構樹狀結構也包含熒幕朗讀程式的ARIA標籤。

## 區段產生器UI

在Experience Platform中使用區段產生器UI建立、編輯及與區段互動時，下列功能可改善協助工具：

* 區段產生器UI可透過鍵盤導覽存取。
* 熒幕助讀程式應能識別標題的標籤標籤，並可宣告標題及其層級。
* 其他輔助技術可以使用正確編碼的標題來顯示大綱或替代檢視，進而變更頁面的視覺顯示。

您現在可以收合或展開區段產生器畫布的左側和右側邊欄，以獲得更多熒幕空間。 此功能特別實用，因為它在200%縮放下提供完整功能功能。

![區段產生器畫布上，左側和右側邊欄公開Widget會反白顯示。](images/left-right-rail-expandables.png)

## 查詢服務編輯器

查詢服務編輯器中提供下列協助工具功能：

* 查詢服務編輯器UI中的顏色對比符合協助工具合規性。
* 編輯器UI外部支援鍵盤導覽。 編輯器UI是內嵌的程式碼映象。

## 來源和目的地中的「系統檢視」索引標籤

瀏覽 **[!UICONTROL 系統檢視]** 在「來源」和「目標」中，下列功能可改善協助工具：

* **標籤** 將焦點設定在第一個來源連線卡
   * **標籤** 再次將焦點放在卡片內部的按鈕上
   * 選取 **輸入** 啟用卡片內的行動號召按鈕
* 選取 **輸入** 也會在右側邊欄中啟動更多詳細資料
   * 啟動右側邊欄時，焦點會設定為該區域。 **標籤** 重點在 **關閉** 右側邊欄窗格的。 選取 **標籤** 再次透過右側欄面板移動焦點
   * 如果有超過一張來源連線卡， **標籤** 在連線中移動
   * 使用 **方向鍵（向上、向下、向左及向右）** 在來源清單中移動
   * 選取 **標籤** 將焦點設定在右側欄面板上
