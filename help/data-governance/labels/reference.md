# 資料使用標籤字彙表 {#data-usage-labels-glossary}

>[!CONTEXTUALHELP]
>id="platform_policies_labeltype"
>title="標籤類型"
>abstract="資料使用量標籤有數個類別。 Adobe定義的標籤包括合約標籤、身分標籤和敏感標籤。 貴組織定義的標籤會分類為自訂標籤。"
>text="See the data usage labels glossary for more information on these label types."

資料使用量標籤可讓您根據套用至該資料的使用量原則，對資料集和欄位進行分類。 Adobe Experience Platform資料控管提供數個現成可用的核心資料使用標籤，供您用來開始分類資料。

本檔案概述目前由 [!DNL Experience Platform]. 如需資料控管的詳細資訊，請參閱 [資料控管概觀](../home.md).

## 合約標籤

合約「C」標籤可用來分類有合約義務或與貴組織資料控管政策相關的資料。

| 標籤 | 定義 |
| --- | --- |
| **C1** | 資料只能以匯總形式從Adobe Experience Cloud匯出，不含個別或裝置識別碼。 [更多資訊...](#c1) |
| **C2** | 資料無法匯出至協力廠商。 [更多資訊...](#c2) |
| **C3** | 資料無法結合，或與可直接識別身分的資訊搭配使用。 [更多資訊...](#c3) |
| **C4** | 資料無法用於定位任何廣告或內容，無論是站上或跨網站。 [更多資訊...](#c4) |
| **C5** | 資料無法用於內容或廣告的興趣型跨網站目標定位。 [更多資訊...](#c5) |
| **C6** | 資料無法用於站上廣告鎖定目標。 [更多資訊...](#c6) |
| **C7** | 資料無法用於內容的站上目標定位。 [更多資訊...](#c7) |
| **C8** | 資料無法用於測量貴組織的網站或應用程式。 [更多資訊...](#c8) |
| **C9** | 資料無法用於資料科學工作流程。 [更多資訊...](#c9) |
| **C10** | 資料無法用於匯整身分啟用。 [更多資訊...](#c10) |
| **C11** | 資料無法與區段比對合作夥伴共用。 [更多資訊...](#c11) |

## 身分標籤

身分「I」標籤可用來分類可識別或聯絡特定人員的資料。

| 標籤 | 定義 |
| --- | --- |
| **I1** | 可識別或聯絡特定人員（而非裝置）的直接識別資料。 |
| **I2** | 可與任何其他資料結合使用，以識別或聯絡特定人員的可間接識別資料。 |

## 敏感標籤 {#sensitive}

敏感「S」標籤可用來將您和您的組織視為敏感的資料分類。

您可能認為敏感的資料類型可能是不同類型的地理資料；不過，此類別不限於地理資料。

| 標籤 | 定義 |
| --- | --- |
| **S1** | 指定經緯度的資料，這些經緯度可用來判斷裝置的精確位置。 |
| **S2** | 可用於確定廣義地理序列區域的資料。 |
| **PSPD** | 允許的敏感個人資料(PSPD)是指在Adobe合約上允許上傳被視為「敏感」、「特殊資料類別」或適用法律所使用類似術語的資料。 這特別不包括受保護的健康資訊(PHI)和其他受管理的健康資料。 |
| **RHD** | 指保護健康資訊(PHI)或與Adobe允許上傳的患者有關的資訊的資料。 |

## 附錄

以下各節提供可用資料使用標籤的其他資訊。

### 合同標籤詳細資訊

以下各節提供與執行特定合同&quot;C&quot;標籤有關的詳細資訊。

#### C1 {#c1}

有些資料只能匯出自Adobe Experience Cloud，但不含個別或裝置識別碼。 例如，源自社交網路的資料。

#### C2 {#c2}

有些資料提供者在其合約中訂有條款，禁止從最初收集到的資料的地方導出資料。 例如，社交網路合約通常會限制您從這些合約收到的資料傳輸。 C2標籤的限制性比 [C1](#c1)，僅需匯總及匿名資料。

#### C3 {#c3}

有些資料提供者在其合約中訂有條款，禁止將該資料與可直接識別資訊合併或使用。 例如，源自廣告網路、廣告伺服器和第三方資料提供者的資料合約通常包含特定合約禁止將這類資料與可直接識別的資料搭配使用。

#### C4 {#c4}

C4是限制最嚴格的標籤，它包含標籤 [C5](#c5), [C6](#c6)，和 [C7](#c7).

#### C5 {#c5}

如果符合下列三個條件，就會發生興趣型鎖定目標或個人化：現場收集的資料用於(1)對用戶的興趣做出推斷，(2)用於其它上下文，例如在其他站點或應用（非現場）上，和(3)用於根據這些推斷來選擇提供哪些內容或廣告。

來自多個網站的資料組合（包括站上資料和離站資料的組合，或來自多個離站來源的資料組合）稱為跨網站資料。 不同的網站代表不同的內容，因此在任何內容中使用跨網站資料的方式都與原始內容不同。 通常會收集並處理跨網站資料，以便對使用者的興趣做出推論。 因此，使用跨網站資料來定位廣告或內容，無論廣告或內容是顯示在網站上還是離站，通常都符合興趣型定位的資格。 例如，如果將站上資料與離站資料結合使用，以選取要在組織自己的網站上顯示使用者的廣告，則該使用將符合興趣型定位。 再舉一例，將廣告重新定位給離站使用者，也可能符合興趣型定位的資格。

單獨使用離站資料進行定位也可能符合興趣型定位，因為離站資料通常會收集並處理，以便對使用者的興趣做出推論。

不過，僅使用站上資料來定位內容或廣告，通常不符合興趣型定位的資格。 不符合興趣型定位之網站上定位會視為兩個不同的標籤。 具體來說，標籤C6可處理站上廣告定位和報告，並具體說明廣告選擇、傳送和報告，標籤C7可處理站上內容選擇、傳送和報告（站上內容定位）。

最終，標籤的解釋以及如何強制使用帶有該標籤的資料取決於您。 以下提供IAB和DAA架構以供參考：

IAB:個人化。 收集和處理關於您使用此服務的資訊，以隨時間為您在其他內容（如其他網站或應用程式）中個人化廣告和/或內容。 通常，網站或應用程式的內容會用來推斷您的興趣，以便為未來的廣告和/或內容選擇提供參考。

資料：線上行為廣告。 從特定電腦或設備收集關於隨時間和跨非附屬網站的Web查看行為的資料，以便使用這些資料來預測用戶偏好或興趣，以基於從這種Web查看行為推斷的偏好或興趣向該電腦或設備提供廣告。

#### C6 {#c6}

廣告是顯示在網站或應用程式上的訊息或通知，包括文字和影像，主要用於促銷商品或服務的銷售。 此類訊息或通知的用途由您決定。 廣告與站上內容不同，由標籤涵蓋 [C7](#c7). 具有C6標籤的資料無法用於站上廣告目標定位，包括在貴組織網站或應用程式上選擇和投放廣告，或用於評估此類廣告的投放和效果。 這包括使用先前收集的關於使用者興趣的現場資料來選取廣告、處理關於顯示哪些廣告、顯示時間和位置的資料，以及使用者是否採取了與廣告相關的任何動作，例如選取廣告或購買。 通常，根據使用者的站上活動來推斷使用者的偏好，然後在站上廣告鎖定目標中使用這些偏好不符合興趣型鎖定目標（也稱為個人化），因為這不符合興趣型鎖定目標所需的全部三個需求。 *[請參閱標籤C5以了解這些要求。](#c5)*

最終，標籤的解釋以及如何強制使用帶有該標籤的資料取決於您。 以下提供IAB和DAA架構以供參考：

IAB:3. 廣告選擇、傳遞、報告：資訊的收集，以及與先前收集的資訊的組合，來選擇和提供廣告，並衡量此類廣告的交付和效果。 這包括使用先前收集到的關於您興趣的資訊來選取廣告、處理關於顯示哪些廣告、顯示頻率、顯示時間和位置的資料，以及您是否採取了與廣告相關的任何動作，包括選取廣告或進行購買。 這不包括個人化，個人化是指收集和處理關於您使用此服務，隨後在其他內容（例如網站或應用程式）中為您個人化廣告和/或內容的資訊。

資料：線上行為廣告不包括第一方、廣告傳遞或廣告報告或情境式廣告的活動（即根據所瀏覽網頁內容、消費者目前造訪網頁或搜尋查詢的廣告）。

#### C7 {#c7}

現場內容是文字和影像，旨在提供資訊、教育或娛樂，並非為了促進商品或服務的銷售而建立。 內容的用途取決於您，包括內容是否符合原生廣告的資格。 C7標籤不是要涵蓋由標籤覆蓋的現場廣告 [C6](#c6). 具有C7標籤的資料無法用於站上內容目標定位，包括在貴組織網站或應用程式上選擇和傳遞內容，或用於測量此類內容的傳遞和有效性。 這包括先前收集的關於使用者在選取內容中的興趣、處理關於顯示內容、顯示頻率或時間、顯示時間和位置的資料，以及使用者是否採取了與內容相關的任何動作（例如選取內容）。 通常，根據使用者的站上活動來推斷使用者的偏好，然後在站上內容鎖定目標中使用這些偏好不符合興趣型鎖定目標（又稱為個人化），因為這不符合興趣型鎖定目標所需的全部三個需求。 *[請參閱標籤C5以了解這些要求。](#c5)*

最終，標籤的解釋以及如何強制使用帶有該標籤的資料取決於您。 以下提供IAB和DAA架構以供參考：

IAB:4. 內容選擇、傳遞、報告：資訊的收集，以及與先前收集的資訊的組合，以選擇和提供內容給您，並衡量此類內容的傳送和有效性。 這包括使用先前收集的關於您興趣的資訊來選取內容、處理關於顯示內容、顯示頻率或時間長度、顯示時間和位置的資料，以及您是否採取了與內容相關的任何動作，包括選取內容。 這不包括個人化，個人化是指收集和處理關於您使用此服務，隨後在其他內容（例如網站或應用程式）中為您個人化內容和/或廣告的資訊。

資料：線上行為廣告不包括第一方、廣告傳遞或廣告報告或情境式廣告的活動（即根據所瀏覽網頁的內容、消費者目前造訪網頁或搜尋查詢的廣告）。

#### C8 {#c8}

資料無法用來測量、了解及報告使用者對您組織網站或應用程式的使用情形。 這不包括興趣型定位（跨網站定位），這是關於您使用此服務，隨後在其他內容（例如網站或應用程式）中為您個人化內容和/或廣告的資訊集合。

#### C9 {#c9}

有些合同明確禁止資料科學使用資料。 有時，這些措辭的措辭禁止將資料用於人工智慧(AI)、機器學習(ML)或建模。

#### C10 {#c10}

有些資料使用原則會限制使用連結的身分資料來進行個人化。 如果區段的合併政策使用「專用圖表」選項，則C10標籤會自動套用至區段。

#### C11 {#c11}

Adobe Experience Platform區段比對可讓您將第一方區段與隱私權和同意偏好設定進行比對，以利擴充設定檔和下游分析。 C11標籤表示不應在 [!DNL Segment Match] 程式。 在您決定要從「區段比對」中排除的資料集和/或欄位，並據此新增C11標籤後，「區段比對」工作流程會自動強制執行標籤。
