# [!UICONTROL 對象]儀表板 {#audience-dashboard}

Adobe Experience Platform使用者介面(UI)提供了一個儀表板，您可以透過該儀表板檢視有關受眾的重要資訊，如每日快照期間所擷取。

如需如何在UI中存取和互動對象儀表板的詳細指示，以及深入瞭解儀表板中顯示的可用量度，請造訪[對象儀表板指南](../../dashboards/guides/audiences.md)。

如需Experience Platform使用者介面中所有儀表板功能的概觀，請先閱讀[儀表板概觀](../../dashboards/home.md)。
