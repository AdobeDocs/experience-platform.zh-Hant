# 重設合併ID

>[!IMPORTANT]
>
>此動作已棄用。 請改用[收集內部連結點按](../configure/data-collection.md#collect-internal-link-clicks)設定。

**[!UICONTROL Reset merge ID]**&#x200B;動作型別可讓您分隔在相同頁面上呼叫的事件。 它通常用於內部連結情況，其中您可能想要將多個裝載傳送至Adobe。 此動作可讓您重設事件的合併ID，以便在送達Edge Network後不會將其視為相同事件的一部分。

如果您想要控制相同頁面上多個事件的分隔或合併方式，請在設定標籤擴充功能時，使用[收集內部連結點按次數](../configure/data-collection.md#collect-internal-link-clicks)選項。
