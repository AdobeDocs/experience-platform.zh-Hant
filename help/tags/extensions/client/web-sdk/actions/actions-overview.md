# 動作類型

此頁面列出標籤規則中所有支援的動作。 若要建立或編輯標籤規則內的動作：

1. 使用您的Adobe ID憑證登入[experience.adobe.com](https://experience.adobe.com)。
1. 導覽至&#x200B;**[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**。
1. 選取所需的標籤屬性。
1. 導覽至&#x200B;**[!UICONTROL Rules]**，然後選取所需的規則。
1. 在[!UICONTROL Actions]下，選取現有動作或建立動作。
1. 將[!UICONTROL Extension]下拉式欄位設定為&#x200B;**[!UICONTROL Adobe Experience Platform Web SDK]**，然後選取所需的[!UICONTROL Action Type]。

Web SDK標籤擴充功能提供下列動作：

* [**[!UICONTROL Apply propositions]**](apply-propositions.md)：在單頁應用程式中轉譯主張，而不增加量度。
* [**[!UICONTROL Apply response]**](apply-response.md)：根據Edge Network的回應執行動作。
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md)：手動觸發規則集評估。
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md)：將舊版Media API匯出至視窗物件。
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md)：允許在您擁有的網域間共用訪客識別碼。
* [**[!UICONTROL Send event]**](send-event.md)：將事件資料傳送至Edge Network。
* [**[!UICONTROL Send media event]**](send-media-event.md)：將媒體資料傳送至Edge Network。
* [**[!UICONTROL Set consent]**](set-consent.md)：設定訪客的所需同意。
* [**[!UICONTROL Update variable]**](update-variable.md)：修改[變數](../data-element-types.md#variable)資料元素。
