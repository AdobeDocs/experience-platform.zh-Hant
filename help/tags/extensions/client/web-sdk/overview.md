# Adobe Experience Platform Web SDK擴充功能概觀

>[!NOTE]
>
>Adobe Experience Platform Launch已經過品牌重塑，現在是Adobe Experience Platform中的一套資料收集技術。 因此，所有產品檔案中出現了幾項術語變更。 請參閱下列[檔案](../../../term-updates.md)，以取得術語變更的彙總參考資料。

Adobe Experience Platform Web SDK擴充功能會透過Adobe Experience CloudEdge Network，將資料從網頁屬性傳送至Adobe Experience Platform。 Platform Web SDK擴充功能可將資料串流至平台、同步身分、選擇加入並自動收集內容資料。

如需有關設定Platform Web SDK擴充功能的詳細資訊，請參閱[設定Web SDK標籤擴充功能](web-sdk-extension-configuration.md)。
