# 建立擴充功能套件zip檔案

若要準備要在Adobe Experience Platform中上傳的擴充功能套件，請參閱[@adobe/reactor-packager](https://www.npmjs.com/package/@adobe/reactor-packager)命令列工具的API檔案。 它會輸出預期格式的已準備zip檔案，以供[上傳副檔名](./upload-and-test.md)。
