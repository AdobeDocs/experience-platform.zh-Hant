# 支援的瀏覽器

Adobe Experience Platform中的標籤執行階段（提供給客戶的網站）與多個瀏覽器相容：

- Chrome (最新)
- Safari (最新)
- Firefox (最新)
- Edge (最新)
- iOS Safari (最新)
- Android Chrome (最新)

執行階段中包含的任何擴充功能程式碼也都應盡可能支援這些瀏覽器。

資料收集UI的設計可支援下列瀏覽器：

- Chrome (最新)
- Safari (最新)
- Firefox (最新)
- Edge (最新)

資料收集UI中顯示的任何擴充功能程式碼也都應儘可能支援這些瀏覽器。
