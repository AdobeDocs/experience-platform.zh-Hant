# 擴充功能端點

在Reactor API中，擴充功能代表[擴充功能套件](./extension-packages.md)的安裝執行個體。 擴充功能可讓[屬性](./properties.md)使用擴充功能套件所定義的功能。 建立[擴充功能](./data-elements.md)和[規則元件](./rule-components.md)時，會利用這些功能。

擴充功能只屬於一個屬性。 屬性可以有許多擴充功能，但特定擴充功能套件最多只能有一個已安裝的執行個體。

## 快速入門

此指南中使用的端點是[Reactor API](https://www.adobe.io/experience-platform-apis/references/reactor/)的一部分。 在繼續之前，請檢閱[快速入門手冊](../getting-started.md)，以取得有關如何向API驗證的重要資訊。

## 擷取擴充功能清單 {#list}

您可以提出GET要求來擷取屬性的擴充功能清單。

**API格式**

```http
GET properties/{PROPERTY_ID}/extensions
```

| 參數 | 說明 |
| --- | --- |
| `{PROPERTY_ID}` | 您要列出其副檔名的屬性之`id`。 |

{style="table-layout:auto"}

>[!NOTE]
>
>您可以使用查詢引數，根據下列屬性篩選列出的擴充功能：<ul><li>`created_at`</li><li>`dirty`</li><li>`display_name`</li><li>`enabled`</li><li>`name`</li><li>`origin_id`</li><li>`published`</li><li>`published_at`</li><li>`revision_number`</li><li>`updated_at`</li><li>`version`</li></ul>如需詳細資訊，請參閱[篩選回應](../guides/filtering.md)的指南。

**要求**

```shell
curl -X GET \
  https://reactor.adobe.io/properties/PRee071cb5b7794f42b74c913e1ad2e325/extensions \
  -H 'Authorization: Bearer {ACCESS_TOKEN}' \
  -H 'x-api-key: {API_KEY}' \
  -H 'x-gw-ims-org-id: {ORG_ID}' \
  -H "Content-Type: application/vnd.api+json" \
  -H 'Accept: application/vnd.api+json;revision=1'
```

**回應**

## 查詢擴充功能 {#lookup}

您可以在GET請求的路徑中提供擴充功能ID，以查詢擴充功能。

>[!NOTE]
>
>刪除擴充功能時，會在系統中將其標示為已刪除，但實際上並不會將其移除。 因此，您可以擷取已刪除的擴充功能。 透過傳回之擴充功能資料的`meta`中是否存在`deleted_at`屬性，可識別已刪除的擴充功能。

**API格式**

```http
GET /extensions/{EXTENSION_ID}
```

| 參數 | 說明 |
| --- | --- |
| `EXTENSION_ID` | 您要查閱之擴充功能的`id`。 |

## 建立或更新擴充功能 {#create}

擴充功能是透過參照[擴充功能套件](./extension-packages.md)並將已安裝的擴充功能新增至屬性來建立。 當安裝工作完成時，會傳回回應，指出擴充功能是否已成功安裝。

**API格式**

```http
POST /properties/{PROPERTY_ID}/extensions
```

| 參數 | 說明 |
| --- | --- |
| `PROPERTY_ID` | 您想要安裝擴充功能之屬性的`id`。 |

{style="table-layout:auto"}

**要求**

```shell
curl -X POST \
  https://reactor.adobe.io/properties/PRee071cb5b7794f42b74c913e1ad2e325/extensions \
  -H 'Authorization: Bearer {ACCESS_TOKEN}' \
  -H 'x-api-key: {API_KEY}' \
  -H 'x-gw-ims-org-id: {ORG_ID}' \
  -H 'Content-Type: application/vnd.api+json' \
  -d '{
        "data": {
          "attributes": {
            "delegate_descriptor_id": "example-package::extensionConfiguration::config",
            "enabled": true,
            "settings": "{\"elementProperty\":\"html\",\"elementSelector\":\".target-element\"}"
          },
          "relationships": {
            "extension_package": {
              "data": {
                "id": "EP75db2452065b44e2b8a38ca883ce369a",
                "type": "extension_packages"
              }
            }
          },
          "type": "extensions"
        }
      }'
```

| 屬性 | 說明 |
| --- | --- |
| `relationships.extension_package` | **（必要）**&#x200B;參考正在安裝的擴充功能套件識別碼的物件。 |
| `attributes.delegate_descriptor_id` | 如果您的擴充功能需要自訂設定，則它也需要委派描述項ID。 如需詳細資訊，請參閱[委派描述項ID](../guides/delegate-descriptor-ids.md)的指南。 |
| `attributes.enabled` | 表示擴充功能是否已啟用的布林值。 |
| `attributes.settings` | 以字串表示的設定JSON物件。 |

{style="table-layout:auto"}

## 修訂擴充功能 {#revise}

您可以在PATCH請求的路徑中包含擴充功能ID來修訂擴充功能。

**API格式**

```http
PATCH /extensions/{EXTENSION_ID}
```

| 參數 | 說明 |
| --- | --- |
| `EXTENSION_ID` | 您要修訂之擴充功能的`id`。 |

{style="table-layout:auto"}

**要求**

與[建立擴充功能](#create)一樣，修訂封裝的本機版本必須透過表單資料上傳。

```shell
curl -X PATCH \
  https://reactor.adobe.io/extensions/EX8ce7ced633f34bd48d33089ff8fad082 \
  -H 'Authorization: Bearer {ACCESS_TOKEN}' \
  -H 'x-api-key: {API_KEY}' \
  -H 'x-gw-ims-org-id: {ORG_ID}' \
  -H 'Content-Type: application/vnd.api+json' \
  -d '{
        "data": {
          "attributes": {
            "enabled": false,
          },
          "meta": {
            "action": "revise"
          },
          "id": "EX85509bc7baea4f8599bc44024ed3f110",
          "type": "extensions"
        }
      }'
```

| 屬性 | 說明 |
| --- | --- |
| `attributes` | 您想要修訂的屬性。 對於擴充功能，您可以修訂其`delegate_descriptor_id`、`enabled`和`settings`屬性。 |
| `meta.action` | 進行修訂時必須包含在`revise`的值中。 |

{style="table-layout:auto"}

## 刪除擴充功能 {#private-release}

您可以在DELETE請求的路徑中包含擴充功能ID以刪除擴充功能。

**API格式**

```http
DELETE /extensions/{EXTENSION_ID}
```

| 參數 | 說明 |
| --- | --- |
| `EXTENSION_ID` | 您要刪除之擴充功能的`id`。 |

{style="table-layout:auto"}

**要求**

```shell
curl -X DELETE \
  https://reactor.adobe.io/extensions/EX8ce7ced633f34bd48d33089ff8fad082 \
  -H 'Authorization: Bearer {ACCESS_TOKEN}' \
  -H 'x-api-key: {API_KEY}' \
  -H 'x-gw-ims-org-id: {ORG_ID}'
```

**回應**

成功的回應會傳回HTTP狀態204 （無內容），但沒有回應內文，這表示擴充功能已刪除。

擴充功能是「重要」資源，這表示您可以在每個個別資源上建立和擷取文字型附註。 如需有關如何管理擴充功能和其他相容資源的附註的詳細資訊，請參閱[附註端點指南](./notes.md)。

### 查詢擴充功能的相關屬性 {#property}

您可以將`/property`附加至GET請求的路徑，以查詢擁有擴充功能的屬性。

**API格式**

```http
GET  /extensions/{EXTENSION_ID}/property
```

| 參數 | 說明 |
| --- | --- |
| `{EXTENSION_ID}` | 您要查詢其屬性的延伸模組`id`。 |

{style="table-layout:auto"}

**要求**

```shell
curl -X GET \
  https://reactor.adobe.io/extensions/EX8ce7ced633f34bd48d33089ff8fad082/property \
  -H 'Authorization: Bearer {ACCESS_TOKEN}' \
  -H 'x-api-key: {API_KEY}' \
  -H 'x-gw-ims-org-id: {ORG_ID}' \
  -H "Content-Type: application/vnd.api+json" \
  -H 'Accept: application/vnd.api+json;revision=1'
```
