# 疑難排解

Adobe Experience Platform邊緣網路伺服器API可讓您透過邊緣網路資料收集管道處理事件時，從服務擷取除錯資訊。

與 [Experience Platform偵錯器](https://experienceleague.adobe.com/docs/debugger-learn/tutorials/experience-platform-debugger/introduction-to-the-experience-platform-debugger.html?lang=en) 可讓您對API型實作除錯。

使用 [格里芬專案](https://aep-sdks.gitbook.io/docs/beta/project-griffon)，您可以建立除錯工作階段ID，以便進一步用於邊緣網路請求以追蹤事件。
