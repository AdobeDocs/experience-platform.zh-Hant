# 資料Distiller套件

Data Distiller是套件，包含Adobe Experience Platform中的一部分功能。 此套件旨在透過在Query Service中執行的批次查詢，為即時客戶設定檔或分析資料使用案例擷取後的資料準備。 請參閱 [資料Distiller概觀](../data-distiller/overview.md) 如需資料Distiller套件的詳細資訊。
