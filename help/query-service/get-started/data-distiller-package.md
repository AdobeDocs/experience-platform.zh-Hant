# 資料Distiller套件

Data Distiller是一套產品，其中包括Adobe Experience Platform的部分功能。 此套件旨在透過在Query Service中執行的批次查詢，為即時客戶設定檔或分析資料使用案例準備擷取後的資料。 請參閱 [資料Distiller概觀](../data-distiller/overview.md) 以取得資料Distiller套件的詳細資訊。
