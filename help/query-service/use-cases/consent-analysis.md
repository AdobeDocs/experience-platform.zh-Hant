# 同意分析和追蹤

在現今的行銷格局中，您必須瞭解並尊重客戶同意偏好設定。 Adobe Real-time Customer Data Platform可讓行銷人員分析客戶同意，以建立信任、遵守隱私權法規，並提供更個人化的體驗。

同意分析指南詳細說明如何針對Real-Time CDP資料的各種行銷使用案例建立同意儀表板。 若要瞭解如何建立同意分析儀表板，以追蹤使用者同意隨時間變化的趨勢，請參閱 [同意分析儀表板指南](../../dashboards/insights-use-cases/consent-analysis.md).
