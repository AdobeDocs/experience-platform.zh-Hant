# [!DNL Spark] SQL函式

Adobe Experience Platform Query Service提供數種內建的Spark SQL函式，以擴充SQL功能。 本檔案列出查詢服務支援的Spark SQL函式。

有關函式的更多詳細資訊，包括其語法、用法和示例，請閱讀[Spark SQL函式文檔](https://spark.apache.org/docs/latest/api/sql/index.html)。

>[!NOTE]
>
>並非外部文檔中的所有功能都受支援。
