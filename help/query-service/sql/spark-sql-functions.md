# [!DNL Spark] SQL函式

Adobe Experience Platform Query Service提供數種內建的Spark SQL函式以擴充SQL功能。 本文檔列出了Query Service支援的Spark SQL函式。

如需函式的詳細資訊，包括其語法、使用方式和範例，請參閱 [Spark SQL函式文檔](https://spark.apache.org/docs/latest/api/sql/index.html).

>[!NOTE]
>
>外部檔案中並非所有函式皆受支援。
