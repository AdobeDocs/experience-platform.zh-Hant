# 查詢服務SQL參考

[!DNL Query Service]提供對`SELECT`敘述句和其他有限命令使用標準ANSI SQL的功能。 [!DNL Query Service]不包含資料定義語言或資料操作語言功能。 透過服務查詢的資料來自[!DNL Experience Data Lake]，其中包含大部分不可變的資料集。
