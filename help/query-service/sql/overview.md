# SQL參考

查詢服務提供對語句和其他有限命令使用標 `SELECT` 準ANSI SQL的能力。 查詢服務不包括「資料定義語言」或「資料操縱語言」功能。 透過服務查詢的資料來自Experience Data Lake，其中包含的資料基本上不可變。
