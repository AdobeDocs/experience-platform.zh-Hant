# SQL參考

[!DNL Query Service] 提供了對語句和其他有限命令使用標 `SELECT` 準ANSI SQL的能力。 [!DNL Query Service] 不包含「資料定義語言」或「資料操縱語言」功能。 通過服務查詢的資料來自於基 [!DNL Experience Data Lake]本不可變的資料集。
