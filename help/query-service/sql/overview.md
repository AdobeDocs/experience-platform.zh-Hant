# 查詢服務SQL引用

[!DNL Query Service] 提供使用標準ANSI SQL的功能 `SELECT` 語句和其他有限命令。 [!DNL Query Service] 不包括資料定義語言或資料操作語言功能。 通過服務查詢的資料來自 [!DNL Experience Data Lake]它包含基本不可變的資料集。
