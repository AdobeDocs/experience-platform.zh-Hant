# 查詢服務SQL參考

[!DNL Query Service] 提供使用標準ANSI SQL的能力 `SELECT` 語句和其他有限命令。 [!DNL Query Service] 不包含資料定義語言或資料操作語言功能。 透過服務查詢的資料來自 [!DNL Experience Data Lake]，其中包含大致不可修改的資料集。
