# 來源概觀

即時客戶資料平台可讓您從外部來源擷取資料，同時讓您能夠使用平台服務來建構、標示並增強傳入資料。 您可以從多種來源收集資料，例如Adobe解決方案、雲端儲存空間、協力廠商軟體和您的CRM。

如需來源的詳細概觀，請閱讀來源 [概觀](../../source-connectors/home.md)。
