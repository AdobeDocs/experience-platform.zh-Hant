# Real-time Customer Data Platform來源

[!DNL Adobe Real-Time Customer Data Platform] 可讓您從外部來源擷取資料，同時可以使用 服務來建立、加標籤及增強傳入資料。 [!DNL Platform]您可以從多種來源(如Adobe解決方案、基於雲的儲存、第三方軟體和您的CRM)中接收資料。

有關源的更詳細的概述，包括可用源的清單和指向教程的連結，請閱讀 [源文檔](../../sources/home.md)。
