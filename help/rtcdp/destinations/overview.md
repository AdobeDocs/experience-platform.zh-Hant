# Real-time Customer Data Platform目的地

目的地是預先建置與目的地平台的整合，可順暢地從Adobe Experience Platform啟動資料。 您可以使用目的地來針對跨通路行銷活動、電子郵件行銷活動、目標廣告和其他許多使用案例，啟用已知和未知的資料。

如需目的地的詳細概觀，包括可用目的地清單和教學課程連結，請參閱 [目的地檔案](../../destinations/home.md).
