# 流服務錯誤消息

流服務用於收集和集中平台內不同來源的客戶資料。 該服務提供用戶介面和REST風格的API，使您可以輕鬆地設定到各種資料提供程式的源連接。 通過這些源連接，您可以驗證第三方系統，設定接收運行時間，並管理資料接收吞吐量。

本文檔提供了有關流服務的錯誤消息、說明和建議解決方案的目錄。
