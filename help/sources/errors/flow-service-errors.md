# 流程服務錯誤訊息

流量服務是用來收集及集中來自Experience Platform內各種不同來源的客戶資料。 此服務提供使用者介面和RESTful API，可讓您輕鬆設定與各種資料提供者的來源連線。 這些來源連線可讓您驗證協力廠商系統、設定擷取執行時間，以及管理資料擷取輸送量。

本檔案提供有關「流程服務」的錯誤訊息、說明和建議解決方案的目錄。
