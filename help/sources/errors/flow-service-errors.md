# 流服務錯誤消息

流量服務用於收集和集中Platform內各種不同來源的客戶資料。 此服務提供使用者介面和RESTful API，讓您輕鬆設定與各種資料提供者的來源連線。 這些來源連線可讓您驗證您的協力廠商系統、設定擷取執行的時間，以及管理資料擷取吞吐量。

本檔案提供有關流程服務的錯誤訊息、說明和建議解決方案的目錄。
