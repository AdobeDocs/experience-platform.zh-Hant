# [!DNL Shopify] 連接器

Adobe Experience Platform可讓您從外部來源擷取資料，同時使用來建構、加標籤及增強傳入資料 [!DNL Platform] 服務。 您可以從多種來源(如Adobe應用程式、雲儲存、資料庫等)內嵌資料。

[!DNL Experience Platform] 支援從電子商務來源擷取資料。 支援電子商務提供者包括 [!DNL Shopify].

以下檔案提供如何連線的資訊 [!DNL Shopify] to [!DNL Platform] 使用API或使用者介面：
