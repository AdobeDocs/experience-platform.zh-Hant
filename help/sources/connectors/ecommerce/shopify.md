# [!DNL Shopify] 連接器

[!DNL Experience Platform] 支援從協力廠商提供者擷取資 **[!UICONTROL eCommerce]** 料。支援&#x200B;**[!UICONTROL eCommerce]**&#x200B;提供者，包括[!DNL Shopify]。
