# [!DNL Deltashare]

您可以使用[!DNL Deltashare]來源聯結器來建立與[!DNL Databricks Delta Sharing]伺服器的安全連線，並在Adobe Experience Platform中建立虛擬資料集。 透過此功能，您可以查詢虛擬資料集、檢查外部表格並將其對應至Experience Data Model (XDM)結構描述，以及在Experience Platform應用程式和服務中運用這些資料集。 透過[!DNL Deltashare]，您可以避免資料集的實體擷取，藉此減少複製、儲存成本及營運開銷。
