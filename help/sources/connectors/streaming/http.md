# HTTP API連接器

>[!IMPORTANT]
>
>HTTP API來源的映射器函式目前僅支援API，不支援使用者介面。

Adobe Experience Platform可讓您從外部來源擷取資料，同時使用來建構、加標籤及增強傳入資料 [!DNL Platform] 服務。 您可以從多種來源(如Adobe應用程式、雲儲存、資料庫等)內嵌資料。

以下檔案提供如何建立HTTP API串流連接器以連線的資訊 [!DNL Platform] 使用API或使用者介面：
