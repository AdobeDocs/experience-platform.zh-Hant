# [!DNL HTTP API]聯結器

Adobe Experience Platform允許從外部來源擷取資料，同時讓您能夠使用[!DNL Platform]服務來建構、加標籤及增強傳入資料。 您可以從多種來源(例如Adobe應用程式、雲端儲存、資料庫和許多其他來源)內嵌資料。

您可以使用[!DNL HTTP API]來源聯結器將您的資料串流到Platform。 [!DNL Data Prep]函式支援[!DNL HTTP API]來源，可讓您將任何不符合XDM的資料對應到符合XDM的資料集。

以下檔案提供有關如何使用API或使用者介面建立HTTP API串流聯結器以與[!DNL Platform]連線的資訊：

- [使用流量服務API建立HTTP串流連線](../../tutorials/api/create/streaming/http.md)

## 使用UI建立HTTP API串流聯結器
