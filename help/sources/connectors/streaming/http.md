# HTTP API連接器

Adobe Experience Platform可讓您從外部來源擷取資料，同時讓您能夠使用[!DNL Platform]服務來建構、標示並增強傳入資料。 您可以從多種來源（例如Adobe應用程式、雲端儲存空間、資料庫等）擷取資料。

以下檔案提供如何建立HTTP API串流連接器，以使用API或使用者介面與[!DNL Platform]連線的資訊：

## 使用API建立HTTP API串流連接器

- [使用Flow Service API建立HTTP串流連線](../../tutorials/api/create/streaming/http.md)

## 使用UI建立HTTP API串流連接器
