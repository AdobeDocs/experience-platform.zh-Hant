# (Beta) [!DNL Oracle Responsys]來源

>[!NOTE]
>
>[!DNL Oracle Responsys]來源是測試版。 如需使用Beta版標籤來源的相關資訊，請參閱[來源概觀](../../home.md#terms-and-conditions)。

Adobe Experience Platform允許從外部來源擷取資料，同時讓您能夠使用[!DNL Experience Platform]服務來建構、加標籤及增強傳入資料。 您可以從多種來源(例如Adobe應用程式、雲端儲存、資料庫和許多其他來源)內嵌資料。

[!DNL Oracle Responsys]是跨管道行銷活動管理平台，可讓您大規模提供進階智慧，並建立符合客戶特定興趣和偏好設定的自訂訊息。

Experience Platform支援從協力廠商行銷自動化系統擷取資料。 對行銷自動化提供者的支援包括[!DNL Oracle Responsys]。

## IP位址允許清單

使用來源聯結器之前，必須將IP位址清單新增至允許清單。 未能將您區域特定的IP位址新增到允許清單可能會導致使用來源時的錯誤或效能不佳。 如需詳細資訊，請參閱[IP位址允許清單](../../ip-address-allow-list.md)頁面。
