# （測試版） [!DNL Oracle Responsys] 來源

>[!NOTE]
>
>此 [!DNL Oracle Responsys] 來源為測試版。 請參閱 [來源概觀](../../home.md#terms-and-conditions) 以取得使用測試版標籤來源的詳細資訊。

Adobe Experience Platform可讓您從外部來源擷取資料，同時使用來建構、加標籤及增強傳入資料 [!DNL Platform] 服務。 您可以從多種來源(如Adobe應用程式、雲儲存、資料庫等)內嵌資料。

[!DNL Oracle Responsys] 是跨通道行銷活動管理平台，可讓您大規模提供進階智慧，並建立符合客戶特定興趣和偏好的自訂訊息。

Experience Platform支援從協力廠商行銷自動化系統擷取資料。 對行銷自動化提供者的支援包括 [!DNL Oracle Responsys].

## IP位址允許清單

使用來源連接器前，必須將IP位址清單新增至允許清單。 若未將您地區專屬的IP位址新增至允許清單，在使用來源時可能會導致錯誤或效能不佳。 請參閱 [IP位址允許清單](../../ip-address-allow-list.md) 頁面以取得詳細資訊。
