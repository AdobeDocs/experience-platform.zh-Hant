# (Beta) [!DNL Generic REST API]

>[!NOTE]
>
>[!DNL Generic REST API]來源是測試版。 如需使用Beta標籤聯結器的詳細資訊，請參閱[來源概觀](../../home.md#terms-and-conditions)。

Adobe Experience Platform允許從外部來源擷取資料，同時讓您能夠使用[!DNL Experience Platform]服務來建構、加標籤及增強傳入資料。 您可以從多種來源(例如Adobe應用程式、雲端儲存、資料庫和許多其他來源)內嵌資料。

Experience Platform支援從通訊協定應用程式擷取資料，包括[!DNL Generic REST API]。

[!DNL Generic REST API]來源可讓您將資料從REST型應用程式帶入Experience Platform。 [!DNL Generic REST API]支援基本驗證和OAuth 2重新整理程式碼型驗證。

## IP位址允許清單

使用來源聯結器之前，必須將IP位址清單新增至允許清單。 未能將您區域特定的IP位址新增到允許清單可能會導致使用來源時的錯誤或效能不佳。 如需詳細資訊，請參閱[IP位址允許清單](../../ip-address-allow-list.md)頁面。
