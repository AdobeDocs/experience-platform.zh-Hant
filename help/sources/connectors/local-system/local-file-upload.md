# 本機檔案上傳連接器

Adobe Experience Platform可讓您從外部來源和本機系統擷取資料，同時使用[!DNL Platform]服務來建構、加標籤及增強傳入資料。 您可以從多種來源(如Adobe應用程式、雲儲存、資料庫等)內嵌資料。

Experience Platform支援從本機系統擷取資料，讓您能透過使用來源連接器將本機檔案帶入Experience Platform。
