# 本機檔案上傳聯結器

Adobe Experience Platform可讓您從外部來源和本機系統擷取資料，同時能夠使用來建構、加標籤和增強傳入資料 [!DNL Platform] 服務。 您可以從多種來源(例如Adobe應用程式、雲端儲存、資料庫和許多其他來源)內嵌資料。

Experience Platform提供從本機系統擷取資料的支援，可讓您使用來源聯結器將本機檔案移轉到Experience Platform。
