# （測試版） [!DNL Apache Hive] ，連接 [!DNL Azure HDInsights] 器

>[!NOTE]
>Apache Hive on Azure HDInsights連接器為beta版。 如需使用 [測試版標籤連接器的詳細資訊](../../home.md#terms-and-conditions) ，請參閱來源概觀。

Adobe Experience Platform可讓您從外部來源擷取資料，同時提供您使用服務建構、標示及增強傳入資料的 [!DNL Platform] 能力。 您可以從多種來源（例如Adobe應用程式、雲端儲存空間、資料庫等）擷取資料。

[!DNL Experience Platform] 提供從第三方資料庫擷取資料的支援。 [!DNL Platform] 可以連接到不同類型的資料庫，如關係型、 NoSQL或資料倉庫。 對資料庫提供者的支援包括 [!DNL Apache Hive] 在 [!DNL Azure HDInsights]。
