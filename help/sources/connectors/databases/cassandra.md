# [!DNL Apache Cassandra] 連接器

Adobe Experience Platform可讓您從外部來源擷取資料，同時讓您能夠使用[!DNL Platform]服務來建構、標示並增強傳入資料。 您可以從多種來源（例如Adobe應用程式、雲端儲存空間、資料庫等）擷取資料。

[!DNL Experience Platform] 提供從第三方資料庫擷取資料的支援。[!DNL Platform] 可以連接到不同類型的資料庫，如關係型、 NoSQL或資料倉庫。支援資料庫提供者，包括[!DNL Apache Cassandra]。

下面的文檔提供了如何使用API將[!DNL Apache Cassandra]連接到[!DNL Platform]的資訊：
