# [!DNL Apache Cassandra]聯結器

Adobe Experience Platform允許從外部來源擷取資料，同時讓您能夠使用[!DNL Experience Platform]服務來建構、加標籤及增強傳入資料。 您可以從多種來源(例如Adobe應用程式、雲端儲存、資料庫和許多其他來源)內嵌資料。

[!DNL Experience Platform]支援從協力廠商資料庫擷取資料。 [!DNL Experience Platform]可以連線到不同型別的資料庫，例如關聯式、NoSQL或資料倉儲。 支援資料庫提供者包括[!DNL Apache Cassandra]。

以下檔案提供有關如何使用API將[!DNL Apache Cassandra]連線到[!DNL Experience Platform]的資訊：
