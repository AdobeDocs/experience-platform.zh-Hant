# Oracle連接器

Adobe Experience Platform為Microsoft、MySQL和Azure等資料庫提供者提供原生連接。 您可以將這些系統的資料匯入平台。

支援不同類型的第三方資料庫，包括關聯式、NoSQL或資料倉庫。 對資料庫提供方的支援包括Oracle。
