# 建立 [!DNL Azure Blob] 或 [!DNL Amazon] UI中的S3源連接

Adobe Experience Platform的源連接器提供了定期接收外部源資料的能力。

雲儲存源可以將您自己的資料 [!DNL Platform] 無需下載、格式化或上載。 所攝取的資料可以格式化為XDM JSON、XDM Parke或分隔。 流程的每個步驟都整合到「源」工作流中。 [!DNL Platform] 允許您從 [!DNL Azure Blob] 和 [!DNL Amazon S3] 批處理。

有關使用 [!DNL Azure Blob] 源連接器，請閱讀 [[!DNL Azure Blob] 源連接器指南](./blob.md)。

有關使用 [!DNL Amazon S3] 源連接器，請閱讀 [[!DNL Amazon S3] 源連接器指南](./blob.md)。
