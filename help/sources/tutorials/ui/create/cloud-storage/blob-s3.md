# 在使用者介面中建立[!DNL Azure Blob]或[!DNL Amazon] S3來源連線

Adobe Experience Platform中的Source聯結器可讓您依排程擷取外部來源資料。

雲端儲存空間來源可以將您自己的資料帶入[!DNL Platform]，而不需要下載、格式化或上傳。 內嵌的資料可以格式化為XDM JSON、XDM Parquet或分隔。 流程的每個步驟都會整合至來源工作流程。 [!DNL Platform]可讓您透過批次從[!DNL Azure Blob]和[!DNL Amazon S3]匯入資料。

如需使用[!DNL Azure Blob]來源聯結器的詳細指示，請閱讀[[!DNL Azure Blob] 來源聯結器指南](./blob.md)。

如需使用[!DNL Amazon S3]來源聯結器的詳細指示，請閱讀[[!DNL Amazon S3] 來源聯結器指南](./blob.md)。
