# IP地址允許清單

您可以通過網路防火牆定義網路訪問控制。 通過指定適當的源IP範圍，您可以允許資料傳輸服務的通信。 在使用源連接器之前，必須將以下IP地址添加到允許清單中。 如果無法將特定於區域的IP地址添加到允許清單，則在使用源時可能會導致錯誤或效能不佳。
