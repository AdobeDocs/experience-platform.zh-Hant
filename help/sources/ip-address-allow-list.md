# IP位址允許清單

您可以通過網路防火牆定義網路訪問控制。 通過指定適當的源IP範圍，您可以允許資料傳輸服務的通信。 使用來源連接器之前，必須將下列IP位址新增至允許清單。 若未將您地區專屬的IP位址新增至允許清單，在使用來源時可能會導致錯誤或效能不佳。
