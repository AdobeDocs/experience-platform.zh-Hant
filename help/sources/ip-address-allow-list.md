# IP位址允許清單

您可以透過網路防火牆定義網路存取控制。 透過指定適當的來源IP範圍，您可以允許資料傳輸服務的流量。 使用來源聯結器之前，必須將下列IP位址新增至允許清單。 使用來源時，若未將您地區專屬的IP位址新增至允許清單，可能會導致錯誤或效能不佳。
