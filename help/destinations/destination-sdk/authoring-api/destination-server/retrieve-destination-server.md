## 檢索目標伺服器配置 {#retrieve}

通過建立 `GET` 請求 `/authoring/destination-servers` 端點。

>[!TIP]
>
>**API終結點**: `platform.adobe.io/data/core/activation/authoring/destination-servers`

**API格式**

使用以下API格式檢索帳戶的所有目標伺服器配置。

```http
GET /authoring/destination-servers
```

使用以下API格式檢索由 `{INSTANCE_ID}` 的下界。

```http
GET /authoring/destination-servers/{INSTANCE_ID}
```

以下兩個請求將檢索IMS組織或特定目標伺服器配置的所有目標伺服器配置，具體取決於您是否通過 `INSTANCE_ID` 中的設定。

選擇下面的每個頁籤以查看相應的負載及其響應。

>[!BEGINTABS]

>[!TAB 檢索所有目標伺服器配置]

以下請求將根據您有權訪問的目標伺服器配置清單 [!DNL IMS Org ID] 和沙盒配置。

+++請求

```shell
curl -X GET https://platform.adobe.io/data/core/activation/authoring/destination-servers \
 -H 'Authorization: Bearer {ACCESS_TOKEN}' \
 -H 'x-gw-ims-org-id: {ORG_ID}' \
 -H 'x-api-key: {API_KEY}' \
 -H 'x-sandbox-name: {SANDBOX_NAME}'
```

+++
