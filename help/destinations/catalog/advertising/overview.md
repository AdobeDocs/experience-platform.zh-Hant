# 廣告目標概述 {#advertising-destinations}

## 總覽 {#overview}

將Adobe Experience Platform連接到廣告平台DSP，以便與這些平台共用已知和假名受眾。

當連接到廣告目標時，您的受眾將作為ID發送到目標平台，在該平台中，他們映射到目標平台已知的ID。
