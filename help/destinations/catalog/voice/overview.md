# 客戶之聲目標概述 {#voice-of-customer-destinations}

## 總覽

語音平台收集和整理來自各種來源的資料，包括語音消息、呼叫資料和聊天對話。 可以過濾和分析整理的資訊，使公司能夠優化其網站以滿足其要求。
