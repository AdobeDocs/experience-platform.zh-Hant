# 客戶之聲目的地概述 {#voice-of-customer-destinations}

## 概覽

語音平台收集和整理來自各種來源的資料，包括語音消息、通話資料和聊天對話。 可以篩選和分析整理的資訊，讓公司能最佳化其網站以符合其需求。
