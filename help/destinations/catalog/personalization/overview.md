# 個人化目的地概觀 {#personalization-destinations}

## 概觀 {#overview}

個人化平台會收集個人行為的資料和深入分析。 這些平台可協助行銷人員導覽匿名資料、根據先前的活動或作為具有類似習慣的受眾的一部分向潛在客戶提供優惠或廣告。
