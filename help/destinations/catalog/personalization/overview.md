# Personalization目的地概觀 {#personalization-destinations}

## 概觀 {#overview}

Personalization平台會從個人行為收集資料和深入分析。 這些平台可協助行銷人員導覽匿名資料，根據先前的活動或作為具有類似習慣的受眾的一部分為潛在客戶提供優惠或廣告。
