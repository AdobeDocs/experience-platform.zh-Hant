# 個性化目標概述 {#personalization-destinations}

## 總覽 {#overview}

個性化平台從單個行為收集資料和洞察力。 這些平台可以幫助營銷人員導航匿名資料，根據先前的活動或作為具有類似習慣的受眾群的一部分向潛在客戶提供優惠或廣告。
