# 個人化目的地概觀 {#personalization-destinations}

## 總覽 {#overview}

個人化平台會收集個別行為的資料和深入分析。 這些平台可協助行銷人員導覽匿名資料、根據先前的活動或作為具有類似習慣之受眾區段的一部分，向潛在客戶提供優惠方案或廣告。

* [Adobe Target](adobe-target-connection.md)
* [（測試版）Adobe Commerce](adobe-commerce.md)
