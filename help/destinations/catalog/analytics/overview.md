# Analytics目的地概觀 {#analytics-destinations}

## 總覽 {#overview}

Analytics目的地會追蹤訪客與您網站上的內容互動的方式，並收集資料以瞭解您的對象行為。 這些平台可針對您最有價值的訪客的參與度和轉換率提供獨特的見解。
