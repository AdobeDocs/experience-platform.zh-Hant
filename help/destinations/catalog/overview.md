# [!DNL Destinations Catalog]

**[!DNL Destinations Catalog]**&#x200B;提供可接收資料的可用目標平台清單。 這些目標平台包括Adobe解決方案、廣告平台、調查平台、電子郵件行銷平台、Experience Platform擴充功能等。
