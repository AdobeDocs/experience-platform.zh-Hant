# 目標目錄概述

目標目錄提供可接收資料的可用目標平台清單。 這些目標平台包括Adobe解決方案、廣告平台、調查平台、電子郵件行銷平台、Experience Platform擴充功能等。

瀏覽目錄以判斷哪些目標平台最適合您的廣告和個人化使用案例，然後設定流程，將您的細分激活至目標。
