# [!UICONTROL 目標] 儀表板 {#destinations-dashboard}

Adobe Experience Platform用戶介面(UI)提供了一個儀表板，您可以通過該儀表板查看有關組織活動目標的重要資訊，這些資訊在每日快照中捕獲。

有關如何訪問和與UI中的目標儀表板交互的詳細說明，以及有關如何瞭解儀表板中顯示的可用度量的詳細資訊，請訪問 [目標儀表板指南](../dashboards/guides/destinations.md)。

有關Experience Platform中所有儀表板功能的概述，請首先閱讀 [儀表板概述](../../dashboards/home.md)。
