# `personalization`

此 `personalization` 物件包含數個屬性，可讓您根據使用者來定製使用者體驗。 在Adobe Target或Adobe Journey Optimizer實作中此功能特別有用，因為這是讓您自訂每位使用者顯示內容的驅動力。

請參閱檔案，瞭解如何 [呈現個人化內容](../../personalization/rendering-personalization-content.md) 以取得詳細資訊。
