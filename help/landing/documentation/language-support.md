# Experience Platform檔案語言支援

Adobe Experience Platform檔案提供多種語言版本。

若要變更顯示檔案的語言，請選取頂端導覽列中的語言圖示。

![](../images/overview/documentation-language.png)

語言下拉式清單開啟時，選擇您要檢視檔案的語言。
