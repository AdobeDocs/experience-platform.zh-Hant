# Experience Platform文檔語言支援

Adobe Experience Platform文檔有多種語言。

要更改文檔顯示的語言，請選擇頂部導航中的語言表徵圖。

![](../images/overview/documentation-language.png)

當語言下拉框開啟時，選擇要查看文檔的語言。

![](../images/overview/documentation-language-select.png)
